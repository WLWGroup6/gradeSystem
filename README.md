# grade_systems
学生成绩的手工录入及查询，是一项非常繁重而枯燥的劳动，耗费许多人力和物力，而且会因人的情绪烦躁而出现饰物，因此在计算机飞速发展的今天，应用数据库技术实现学生成绩的计算机管理是可行而必要的工作。本项目旨在对学生成绩实现计算机管理，提高工作效率，提高管理水平。
本系统的目标如下：
建立学生成绩管理系统，采用计算机对学生成绩进行管理，进一步提高办学效率和现代化水平。需要达到的要求如下：
（1）新用户可以在线注册；
（2）登入时识别用户属性，分为学生页面和教师页面；
（3）在学生页面，可以实现查询成绩、查询个人信息、修改密码等功能；
（4）在教师页面，可以实现学生成绩的查询和修改，学生重修和补考的管理，个人信息和密码的修改等功能；
